
# binary

The binary module handles interacting with the binary/bytes type
## Functions
### len(input)

Returns the number of bytes in a`binary`

Returns a `integer`

### from_bytes(input)

Turns an `array` of bytes into a `binary`

Returns a `binary`

### into_bytes(input)

Turns a `binary` into an `array` of bytes

Returns an `array`
